# CodeJet
CodeJet: A robust code compiler built using React, Express, and Node.js.

# Features 
1) Multi-Language Support: Execute and compile code in various programming languages.
2) Responsive Design: User-friendly interface adaptable to different devices.
3) Fast Execution: Quick compilation and running of code snippets.
4) Error Handling: Clear display of errors or warnings during compilation.
5) Text Mode: A simplified input mode for quick code entry.
6) Syntax Highlighting: Enhances code readability with syntax highlighting.

# Usage:
1) Select Language: Choose from a list of supported programming languages.
2) Code Input: Input code directly into the editor or use text mode for a streamlined experience.
3) Compile & Run: Execute code with a simple click, view results and potential errors.
4) Save & Share: Option to save code snippets or share them with others.

# Technologies Used:
1) React
2) Express
3) Node.js
